NightMode.js

Specify CSS that will only be applied when the sun is down.
I wouldn't recommend using it yet though.

View on Github

Initializing NightMode.js

nightmode.css = {
    "body": {
        "background": "#020111",
        "color": "rgb(215, 215, 215)"
    },
    "body>p": {
        "color": "#93a1a1"
    }
}

Loading CSS

When loading the night mode css you can either use

nightmode.load();

This will only load the night mode css if the sun has set OR You can use

nightmode.changeCSS();

This will just apply the night mode css
